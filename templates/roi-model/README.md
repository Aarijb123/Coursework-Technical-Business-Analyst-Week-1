# ROI model starter tabs

This folder contains CSV files you can import into Excel or Google Sheets as separate tabs.

## Files in this folder

- `01-assumptions.csv` - blank-plus-example assumptions sheet
- `02-baseline-metrics.csv` - baseline operational metrics starter
- `03-opportunities.csv` - opportunity input table
- `04-roi-summary.csv` - output summary format

## How to use them

1. Import each CSV into a workbook as its own tab.
2. Rename tabs if your tool requires it.
3. Add formulas directly in the spreadsheet after import.
4. Keep a note of any assumptions you change from the defaults.

## Important

These are scaffolds only. They should help the workbook function, but they do not contain a finished solution.
